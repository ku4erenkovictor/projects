JetCoders Custom Form

Для подключения кастомных форм, у вас на странице обязательно должен быть подключен jQuery последней версии и в файл jquery.main.js нужно добавть запуск кастомных форм и сам плагин. Способы запуска и работа с кастомными формами описана далее в этом документе.

Виды запуска плагина

Запуск плагина для элементов select.customSelect, input.customRadio, input.customCheckbox :

jQuery('form').customForm();

в данном случае плагин сначала находит все елементы "form" и уже после ищет внутри этих элементов select.customSelect, input.customRadio, input.customCheckbox

jQuery.customForm();

в таком варианте плагин ищет во всем документе элементы select.customSelect, input.customRadio, input.customCheckbox

Опции

В плагине есть опции которые запускаются поумолчанию (т.е. если вы запустили плагин как указано выше, то эти упции тоже запускаются) :

jQuery('form').customForm();

равносильно :

jQuery('form').customForm({
	select: {							// опция отвечает за структуру кастомного селекта
		elements: 'select.customSelect',
		structure: '<div class="selectArea"><a href="#" class="selectButton"><span class="center"></span>'+
				'<span class="right"> </span></a><div class="disabled"></div></div>',
		text: '.center',
		btn: '.selectButton',
		optStructure: '<div class="selectOptions"><ul></ul></div>',
		topClass: 'position-top',
		maxHeight: false,					// в этой опции можно указать максимальную высоту выпадающего меню
		optList: 'ul'
	},
	radio: {							// опция отвечает за структуру кастомного радиобокса
		elements: 'input.customRadio',
		structure: '<div></div>',
		defaultArea: 'radioArea',
		checked: 'radioAreaChecked'
	},
	checkbox: {							// опция отвечает за структуру кастомного чекбокса
		elements: 'input.customCheckbox',
		structure: '<div></div>',
		defaultArea: 'checkboxArea',
		checked: 'checkboxAreaChecked'
	},
	disabled: 'disabled',						// клас для disabled="disabled"
	hoverClass: 'hover'						// клас для включенного селекта
});

Все эти опции можно менять как по отдельности так и все сразу (при этом соблюдая структуру вложенности):

jQuery('form').customForm({			// в данном случае кастомные формы запустятся для select.customSelectOther, 
	select: {				// input.customRadioOther и input.customCheckboxOther
		elements: 'select.customSelectOther'
	},
	radio: {
		elements: 'input.customRadioOther'
	},
	checkbox: {
		elements: 'input.customCheckboxOther'
	}
});

Методы

"destroy"

Метод "destroy" приминяется для отключения кастомных форм (так же удаляется структура и все события). Ниже написан пример который отключает кастомные формы для все элементов внутри "form" :

jQuery('form').customForm("destroy");

Следующий пример отключает кастомный селект с классом .customSelect2 :

jQuery('select.customSelect2').customForm("destroy");

Пример

"refresh"

Метод "refresh" приминяется для перезапуска кастомных форм. Ниже написан пример который перезапускает кастомные формы для все элементов внутри "form" :

jQuery('form').customForm("refresh");

Следующий пример перезапускает кастомный селект с классом .customSelect2 :

jQuery('select.customSelect2').customForm("refresh");

Подключение кастомного скрола для кастомных форм

Подключаем к странице js customScroll и запускаем после создания кастомного селекта:

jQuery.customForm({
	select:{
		elements: 'select.customSelect3',
		optStructure: '<div class="selectOptions"><ul class="custom-scroll"></ul></div>',	// указываем для <ul> класс "custom-scroll"
		maxHeight: 100								// не забываем указывать высоту данного кастомного скролла
	}
});

$('.custom-scroll').customScrollV();
