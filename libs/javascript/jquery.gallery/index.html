JetCoders Gallery

Значения по умолчанию:

Option Description Data Type Default
infinite Включение бесконечной галлереи (только для слайд) Boolean false
activeClass Класс активного элемента String 'active'
duration Время смены елемента Number 300
slideElement Количество листаемых элементов Number 1
effect Выбор между слайд и фейд галереями Boolean, String false (слайд), true (фейд), 'transparent' (фейд без фона)
elements Указываем элементы галлереи String 'ul:eq(0) > li' (По умолчанию первая ul)
switcher Указываем свитчеры String '.switcher > li'
disableBtn Класс для отключения prev и next Boolean or String false (отключено)
prevBtn Указываем кнопку prev String 'a.link-prev, a.btn-prev, .prev'
nextBtn Указываем кнопку next String 'a.link-next, a.btn-next, .next'
circle Возможность возвращаться в начало(конец) галлереи Boolean true
direction Направление слайд галлереи (вертикальное или горизонтальное) Boolean false (горизонтальное)
event Событие мышки ('click', 'mousedown', 'mouseup', 'mouseover') String 'click'
autoHeight Анимирование высоты галереи (только фейд) Boolean false
flexible Перестроение галлереи для адаптива (3 > 2 > 1) Boolean false
oneSlide Ризиновая галарея на один слайд (на всю ширину холдера) Boolean false
autoDetect Автоматическое определение слайд или фейд Boolean true
touch Обработка тач событий Boolean true (включена)
onChange Метод срабатывает при смене елемента Function null

Слайд галереи:

JS: $('.slide1').gallery();

⇐ ⇒

Авторотация (7сек), смена слайдов 1сек и листаем по 2 элемента:

	$('.slide2').gallery({
		autoRotation: 7000,
		duration: 1000,
		slideElement: 2
	});
⇐ ⇒

Свичеры + бесконечная:

	$('.slide3').gallery({
		infinite: true,
		switcher: "div.switcher > span"
	});
⇐ ⇒

Свичеры + Слайд на всю ширину контейнера + Отключение кнопок:

	$('.slide4').gallery({
		oneSlide: true,
		circle: false,
		disableBtn: 'disabled',
		switcher: "div.switcher > span"
	});
⇐ ⇒

Flexible:

При flexible нельзя задавать боковые padding, margin или border на теги <li>. Отступы делаем внутри <li>

	$('.slide5').gallery({
		flexible: true
	});
⇐ ⇒

Фейд галереи:

Все фейд галереи по умолчанию бесконечные.

JS: $('.fade1').gallery();
