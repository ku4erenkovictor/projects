JetCoders Tabs + Steps

Значения по умолчанию:

Option Description Data Type Default
hiddenClass Класс закрытого таба String "hidden"
visibleClass Класс открытого таба String "visible"
activeTab Класс активной ссылки таба String "active"
disabledTab Класс отключенной ссылки таба String "disabled"
steps Если нужны степы, указываем блок с ссылками для степов (Пример) Boolean, String false (отключен)

Методы:

Method Description Example
'setTab' Установки активного таба $('.tabs').tabs('setTab', '#tab01')

Tabs:

Запуск в JS для всех видов табов один: $('.tabs').tabs();

Если не указать класс active то все табы будут скрыты.


Стандартные табы (с отключенной кнопкой):

<div class="wrap tabs">
	<ul class="list">
		<li data-tab="#tab01">Tab 1</li>
		<li data-tab="#tab02" class="active">Tab 2</li>
		<li data-tab="#tab03" class="disabled">Tab 3 (disabled)</li>
	</ul>
	<div class="tab" id="tab01">
		<h3>Tab 1</h3>
	</div>
	<div class="tab" id="tab02">
		<h3>Tab 2</h3>
	</div>
	<div class="tab" id="tab03">
		<h3>Tab 3</h3>
	</div>
</div>

    steps: ".list"
});
