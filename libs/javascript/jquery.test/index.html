Виды запуска плагина

Для начала создадим модальное окно и положем его в конец <body>:

HTML:
		
<div class="modal" id="modal01">
	<span class="close">close</span>
	<p>Текст</p>
</div>

CSS:

.modal{
	width:500px;
	padding:30px 20px;
	border: 1px solid #000;
	position:absolute;
	background:#fff;
	left:-9999px;
	top:-9999px;
	z-index:1000;
}

обязательно задаем id, т.к. javascript будет искать модальное окно по нашему id. Далее создаем в документе кнопку для запуска, у которой в href указываем наш id:

<a class="simplebox" href="#modal01">Open modal</a> 

или

<button class="simplebox" type="button" data-href="#modal01">Open modal</button>

далее запускаем сам плагин:

$('.simplebox').simplebox();

или, если нужно открыть модальное окно без кнопки:

$.simplebox('#modal01');

Опции

В плагине есть опции которые запускаются поумолчанию (т.е. если вы запустили плагин как указано выше, то эти упции тоже запускаются) :

$('.simplebox').simplebox();

или

$.simplebox('#modal01');

равносильно :

$('.simplebox').simplebox({
	duration: 300,
	linkClose: '.close, .btn-close',
	disableClass: 'disabled',
	overlay: {
		box: 'simplebox-overlay',
		color: 'black',
		closeClick: true,
		opacity: 0.3
	},
	positionFrom: false
});

или

$.simplebox('#modal01', {
	duration: 300,
	linkClose: '.close, .btn-close',
	disableClass: 'disabled',
	overlay: {
		box: 'simplebox-overlay',
		color: 'black',
		closeClick: true,
		opacity: 0.3
	},
	positionFrom: false
});

Все эти опции можно менять как по отдельности так и все сразу (при этом соблюдая структуру вложенности):

Опция Значение Описание
duration 300 Время в мс для анимации
linkClose '.close, .btn-close' Элементы которые находятся внутри нашего модального окна, при нажатии на которые будет закрыто модальное окно
disableClass 'disabled' Класс который при необходимости можно задать для <a class="simplebox" href="#modal01">Open modal</a> что бы при клике не открывалось окно.
overlay {object} Опции для overlay (если указать false - оверлей не будет создан)
box 'simplebox-overlay' Класс для overlay
color 'black' Цвет для overlay
closeClick true Закрытие по клику на overlay
opacity 0.3 Прозрачность для overlay
positionFrom false Позиционирование окна от определенного элемента (пример: '#block')

Методы

"$.simplebox.close()"

Метод "$.simplebox.close()" глобальный и закрывает любой открытый simplebox:

jQuery('a.close').click(function(){
	$.simplebox.close();
	return false;
});

События

Событие Описание
onOpen: function(obj){...} Метод запускается перед открытием окна, в obj передается jQuery обьект окна.
afterOpen: function(obj){...} Метод запускается после открытия окна, в obj передается jQuery обьект окна.
onClose: function(obj){...} Метод запускается перед закрытием окна, в obj передается jQuery обьект окна.
afterClose: function(obj){...} Метод запускается после закрытия окна, в obj передается jQuery обьект окна.

Все события обьявляются при запуске плагина, как опции:

$('.simplebox').simplebox({
	onOpen: function(obj){console.log('onOpen', obj);},
	afterOpen: function(obj){console.log('afterOpen', obj);},
	onClose: function(obj){console.log('onClose', obj);},
	afterClose: function(obj){console.log('afterClose', obj);}
});
